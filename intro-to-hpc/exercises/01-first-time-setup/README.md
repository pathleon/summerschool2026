# Exercise: First time setup on a supercomputer

In this exercise we prepare our workspace on LUMI and/or Mahti.

## Tasks

1. Log in to LUMI via ssh as described in
   the general instructions for [LUMI](../../../README_LUMI.md).

2. Prepare your personal working directory under the project scratch area on LUMI
   as described in the general instructions.

3. Test that your favorite code editor works on LUMI.
   See the general instructions for suggestions for editors.

4. (Bonus) Repeat the steps above for Mahti, following
   the general instructions for [Mahti](../../../README_Mahti.md).
